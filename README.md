# Test_repos
Test repository
ここではこのリポジトリの説明を書く

## ローカル環境で編集してみたよ！

ローカル環境にリポジトリをクローン

```rb
git clone [githubのURL]
```

ブランチの確認

```rb
git branch
```

作業スペース（ブランチ）を作成。今回は一人なのでいらないかもだけどクセづけておこう！

```rb
git branch work/001
```

作業スペースへ移動

```rb
git checkout work/001
```

移動の確認

```rb
git branch
```

作業を行う
